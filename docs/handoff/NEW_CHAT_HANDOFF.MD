## 4. 지금 남은 핵심 문제
통합과학1은 3번 추천 개념 순서는 1차 맞췄지만, **4번 후속 연계축의 키워드 분화가 아직 덜 잡혔다.**

### 최우선 검수 3개
1. `과학의 측정과 우리 사회 + 온도 센서`
   - 목표: **물리·시스템 해석 축** 1순위
2. `과학의 측정과 우리 사회 + 미세먼지 농도`
   - 목표: **지구·환경 데이터 해석 축** 1순위
3. `자연 세계의 시간과 공간 + 물 분자`
   - 목표: **물리·규모 해석 축** 1순위
   - 주의: 지구·우주 관측 축이 다시 1위로 올라오면 안 됨
