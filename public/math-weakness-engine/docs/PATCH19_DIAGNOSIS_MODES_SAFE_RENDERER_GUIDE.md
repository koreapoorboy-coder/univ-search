# Patch19 Guide

## 목적

학생이 올린 자료가 개념정리인지, 실제 문제풀이인지, 재제출 답안인지에 따라 다른 진단 결과를 보여줍니다.

## 학생 화면

1. 오늘의 진단 결과
2. 지금 문제점
3. 연결되는 단원
4. 오늘 풀 10문항
5. 학생용 PDF 문제지 열기/저장

## 풀이 과정 진단 기준

- 어느 줄에서 풀이가 틀어졌는가
- 조건을 식으로 정확히 바꿨는가
- 계산 전개가 논리적으로 이어지는가
- 답이 원래 조건과 정의역을 만족하는가
- 같은 유형에서 다시 확인해야 할 개념은 무엇인가

## 오류 수정

`MathHybridReportRenderer.renderTeacherDiagnostics is not a function` 오류가 발생하지 않도록 hybrid.html에서 renderer 함수를 guard 처리했습니다.
