# Patch20 Topic-aware Verification Guide

## 목적

10문항의 구조는 유지하되, 특정 예시(유리수·무리수)가 모든 단원에 고정 출력되지 않도록 주제별 라우팅을 적용한다.

## 주제 라우팅

1. 거듭제곱근/유리수 지수 감지
   - 거듭제곱근, n제곱근, 세제곱근, 네제곱근, 유리수 지수, 분수 지수, 지수법칙
   - 출력: 거듭제곱근 전용 진단 + 10문항

2. 유리수·무리수 감지
   - 유리수, 무리수, 순환소수, 비순환, 유한소수, 무한소수, 분수 꼴
   - 출력: 유리수·무리수 전용 진단 + 10문항

3. 그 외 단원
   - 해당 conceptName 기반 generic proof set 출력

## 검수 포인트

- 대수 거듭제곱근 파일에서 0.333..., 0.121212..., √4/√2 비교 문제가 나오면 실패
- 거듭제곱근 파일에서는 짝수/홀수 근, 음수 밑, 주값, x^n=a, a^(m/n)이 나와야 함
