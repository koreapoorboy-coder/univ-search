# Patch17 Precise Math Connection Output

학생 상단 진단을 일반 코칭 문장이 아니라 수학적으로 정확한 지식 전달 구조로 수정합니다.

## 핵심 변경

- `핵심 개념`, `다음 학년 핵심 단원`, `서술형·융합형 문제` 같은 추상 표현을 학생 화면에서 제거
- 유리수·무리수 자료는 `정수 a, b에 대해 a/b 꼴로 나타낼 수 있는가`를 한 줄 진단에 직접 표시
- 문제점은 최대 3개로 유지하되 수학 기준으로 구체화
  1. 유리수 판단 기준 불명확
  2. 순환소수가 왜 유리수인지 증명 부족
  3. 루트가 있으면 모두 무리수라고 착각할 가능성
- 연결 단원은 정확한 단원명으로 표시
  - 중2: 유리수와 순환소수
  - 중3: 제곱근과 실수
  - 고등: 방정식·부등식, 함수의 정의역
- 10문항 증명형 확인 구조는 유지
