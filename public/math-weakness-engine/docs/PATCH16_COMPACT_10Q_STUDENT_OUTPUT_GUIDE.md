# Patch16 · 학생용 간결 진단 + 10문항 증명 확인

## 목적

학생 화면에서 광범위한 분석 데이터를 먼저 보여주지 않는다. 학생에게는 핵심 문제점과 지금 배우는 개념이 어디에 연결되는지만 짧게 보여주고, 실제 이해 여부는 10문항 증명형 문제로 확인한다.

## 학생 화면 순서

1. 오늘 한 줄 진단
2. 지금 문제점 최대 3개
3. 이 과정이 나중에 연결되는 곳
4. 오늘 풀 10문항
5. 정답/모범답안 접힘 확인
6. 교사용 상세 진단 접힘 처리

## 유리수·무리수 10문항 기본 구조

1. 0.5 유리수 증명
2. -3 유리수 증명
3. 0.333... 유리수 증명
4. 0.121212... 유리수 증명
5. 0.333...이 무리수가 아닌 이유
6. √4가 무리수가 아닌 이유
7. √9가 무리수가 아닌 이유
8. √2가 무리수인 이유
9. “끝나지 않는 소수는 모두 무리수이다” 반례
10. √4와 √2 비교 설명

## 통과 기준

10문항 중 7문항 이상 통과를 기본 기준으로 한다. 단, 반례·비예시·비교 설명 문항을 2개 이상 틀리면 암기형 이해로 판정한다.
