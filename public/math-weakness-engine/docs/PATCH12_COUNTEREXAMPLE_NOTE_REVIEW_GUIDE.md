# Patch 12 · 반례 기반 개념정리/인강필기 검수 모드

## 목적
학생이 인강을 듣고 작성한 개념정리나 필기 자료가 단순 복사인지, 실제 이해가 있는지 구분합니다.

## 핵심 변경
- 개념정리 검수에 반례/비예시 항목 추가
- 조건 없이 공식만 외운 정리를 위험 신호로 표시
- 학생 재정리 틀을 `정의 → 사용 조건 → 대표 예시 → 반례/비예시 → 자주 하는 착각 → 문제 적용 기준` 순서로 고정
- 검수 문항 생성 시 반례 생성 또는 비예시 구분 문항을 포함
