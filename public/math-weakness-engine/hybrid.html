MATHEMATICS DIAGNOSTIC PROGRAM
수학 취약유형 진단

학생의 문제와 풀이를 읽고 취약 개념, 선행 원인, 다음 학습 연결을 진단합니다.

∑
학생 진단 관리자 테스트 화면
진단과 보강은 분리됩니다. 먼저 1차 진단 결과를 확인한 뒤, 필요한 경우에만 보강 10문항과 학생용 PDF를 생성합니다.

1 학생 정보 및 진단 자료

이름과 학년을 입력하고 문제·풀이 PDF 또는 이미지를 업로드하세요. 1차 진단에서는 취약 개념과 연결 단원만 확인하며 보강 문제는 자동으로 만들지 않습니다.

잘 모르겠으면 자동 판별을 그대로 두면 됩니다. 고등학생이 본인 풀이를 올릴 때는 “풀이 과정 진단”을 선택하면 오류 위치와 필요한 개념을 우선 확인합니다.

PDF 또는 사진 파일을 올리면 됩니다. 여러 장이면 한 번에 선택하세요. 개념정리 노트, 오답노트, 시험지, 학생 풀이 사진 모두 가능합니다.

학생 안내 문구

파일에 문제 번호, 풀이 흔적, 오답 표시, 개념 정의, 예시, 자기 설명이 보일수록 진단이 정확해집니다. 1차 진단 후 문제가 필요한 경우에만 보강 문제 생성 버튼을 누르세요.

관리자 연결 설정 열기

학생은 이 영역을 사용할 필요가 없습니다. 기본 Worker 주소가 자동 적용됩니다. 연결 오류가 있을 때만 교사가 확인합니다.
