Worker 연결

OpenAI API Key는 브라우저에 넣지 않습니다. Cloudflare Worker URL만 입력합니다. 저장하면 이 브라우저에만 보관됩니다.

연결 상태

대기Worker URL을 입력하고 URL 저장을 누르세요.
저장 후 새로고침해도 URL이 다시 들어오면 정상입니다.

  

  

    상태 기준
    
health에서 hasOpenAIKey=true, stubMode=false가 나오면 실제 OpenAI 연결 준비가 된 상태입니다. hasOpenAIKey=true만 보여도 API Key Secret 연결은 성공입니다.

  





1. 학생 자료 입력



2. 학생 검수 답안 입력
1차 분석 후 나온 검수 문항을 학생에게 풀리고, 답안을 여기에 붙여넣습니다.
