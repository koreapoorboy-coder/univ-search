수학 하이브리드 AI 진단

학생은 이름·학년·파일만 제출합니다. AI가 먼저 자료를 읽고, 엔진이 단원·개념을 확정한 뒤, AI가 그 결과 안에서만 학생용 문제점·연결 단원·PDF 문항을 작성합니다.

1 학생 진단 자료 제출

학생은 아래 3가지만 입력하면 됩니다. 파일이 시험지/풀이인지, 개념정리/필기인지, 오답노트인지도 AI가 먼저 자동 판별합니다. 개념정리는 상단 설명을 길게 보여주지 않고 문제점·연결 과정만 짧게 제시한 뒤 10문항으로 검수합니다.

잘 모르겠으면 자동 판별을 그대로 두면 됩니다. 고등학생이 본인 풀이를 올릴 때는 “풀이 과정 진단”을 선택하면 오류 위치와 필요한 개념을 우선 확인합니다.

PDF 또는 사진 파일을 올리면 됩니다. 여러 장이면 한 번에 선택하세요. 개념정리 노트, 오답노트, 시험지, 학생 풀이 사진 모두 가능합니다.

학생 안내 문구

파일에 문제 번호, 풀이 흔적, 오답 표시, 개념 정의, 예시, 자기 설명이 보일수록 진단이 정확해집니다. 개념정리는 증명형 10문항으로, 실제 풀이는 오류 위치·개념 연결·보강 10문항으로 확인합니다.

관리자 연결 설정 열기

학생은 이 영역을 사용할 필요가 없습니다. 기본 Worker 주소가 자동 적용됩니다. 연결 오류가 있을 때만 교사가 확인합니다.

기본 Worker URL

현재 기본값은 https://math-diagnosis-worker.koreapoorboy.workers.dev 입니다. 새 브라우저나 학생 기기에서도 자동으로 이 주소를 사용합니다.

연결 상태

자동 적용기본 Worker URL을 사용합니다.
학생은 URL 저장이나 Worker 확인을 누를 필요가 없습니다.
