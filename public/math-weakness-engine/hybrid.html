수학 하이브리드 AI 진단

학생은 이름·학년·파일만 제출합니다. Worker 연결은 자동 적용되며, AI가 파일이 문제풀이인지, 오답노트인지, 개념정리인지 먼저 구분한 뒤 수학 취약유형 엔진과 연결합니다.

1 학생 진단 자료 제출

학생은 아래 3가지만 입력하면 됩니다. 파일이 시험지/풀이인지, 개념정리/필기인지, 오답노트인지도 AI가 먼저 자동 판별합니다. 개념정리는 반례/비예시까지 포함되어 있는지 함께 검수합니다.

PDF 또는 사진 파일을 올리면 됩니다. 여러 장이면 한 번에 선택하세요. 개념정리 노트, 오답노트, 시험지, 학생 풀이 사진 모두 가능합니다.

학생 안내 문구

파일에 문제 번호, 풀이 흔적, 오답 표시, 개념 정의, 예시, 자기 설명이 보일수록 진단이 정확해집니다. 개념정리 파일은 공식만 있는지, 조건·예시·이유·반례/비예시까지 있는지를 따로 검수합니다.

관리자 연결 설정 열기

학생은 이 영역을 사용할 필요가 없습니다. 기본 Worker 주소가 자동 적용됩니다. 연결 오류가 있을 때만 교사가 확인합니다.

기본 Worker URL

현재 기본값은 https://math-diagnosis-worker.koreapoorboy.workers.dev 입니다. 새 브라우저나 학생 기기에서도 자동으로 이 주소를 사용합니다.

연결 상태

자동 적용기본 Worker URL을 사용합니다.
학생은 URL 저장이나 Worker 확인을 누를 필요가 없습니다.
