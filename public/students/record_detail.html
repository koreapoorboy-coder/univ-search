생활기록부 핵심 진단

이 학생의 생활기록부는 근거를 바탕으로 해석하고 구조화한 뒤 설명으로 연결하는 수행 방식이 여러 교과에서 반복적으로 드러나는 구조다.
국어·사회 계열에서는 자료와 텍스트를 읽고 의미를 해석하는 수행이 확인된다.
수학에서는 개념을 논리적으로 전개하고 구조를 설명하는 흐름이 확인된다.
과학·정보·기술가정 계열에서는 원리를 이해한 뒤 자료 해석, 설계, 적용으로 연결하는 태도가 확인된다.

생기부 구조와 연결성

교과별 반복 수행 방식
교과마다 주제는 다르지만 수행 방식은 일정하다. 읽기, 해석, 구조화, 설명, 적용의 흐름이 반복된다.
학생부의 중심 축
이 생활기록부의 중심 축은 분석 → 구조화 → 설명/적용이다. 여러 교과가 하나의 수행 방식으로 묶인다.
구성 해석
이 학생부는 과목별 활동이 흩어진 구조가 아니라, 근거와 원리를 중심으로 사고를 전개하는 학생이라는 점을 일관되게 보여주는 구조다.

데이터 분석 결과

강점
활동 수보다 사고 과정이 드러난다. 무엇을 했는지보다 왜 그렇게 해석했고 어떻게 설명했는지가 확인된다.
한계
자기 판단 기준의 명확성과 독자적 문제 설정은 아직 더 보강해야 한다.
종합 판단
수행의 결은 좋고 연결성도 분명하다. 다만 강점을 더 선명하게 구조화하면 학생부의 완성도가 더 높아진다.
과목별 근거 카드
세특 핵심 근거
작품 구절과 표현을 근거로 인물의 감정 변화와 정서 흐름을 해석한 기록이 확인된다.
기록에서 확인되는 수행
내용 요약이 아니라 의미를 근거와 함께 설명하는 수행이 확인된다.
해석 포인트
국어 교과에서 중요한 근거 기반 해석과 의미 구조화 능력이 분명하게 드러난다.
데이터 분석 결과 1
국어 학업 수행은 감상형보다 분석형으로 확정된다.
데이터 분석 결과 2
세특은 활동 나열형이 아니라 근거와 설명이 드러나는 구조로 판단된다.
데이터 분석 결과 3
학생부 전체에서 분석 → 구조화 → 설명 축의 출발점 역할을 한다.
평가상 의미
읽고 해석하고 설명하는 학생으로 판단된다.
다음 보완 지점
판단 기준을 더 선명하게 드러내면 세특의 밀도가 더 올라간다.
세특 핵심 근거
문제 풀이 과정과 개념 간 연결을 설명하는 기록이 확인된다.
기록에서 확인되는 수행
정답 도출보다 풀이 과정의 논리와 구조 설명이 확인된다.
해석 포인트

평가상 의미

기록에서 확인되는 수행
원인을 설명하고 관련 개념을 연결하는 수행이 확인된다.
