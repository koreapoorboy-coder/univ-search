대학 정시 검색기

배부용 조회 전용 화면입니다. 대학명·학과명 검색, 연도/계열/대학/방식 필터와 학생 백분위/영어등급 비교 판정을 함께 볼 수 있습니다.
학생 입력 (국·수·탐 = 백분위 / 영어 = 등급)
학생 입력: 미입력 판정 기준: 국수탐 최소 여유 + 영어 등급 반영
• 국어/수학/탐1/탐2는 학생 백분위 - 대학 컷으로 비교합니다.
• 영어는 학생 영어등급 ≤ 대학 영어컷등급이면 충족입니다.
• 기본 판정: 최소 여유가 +5 이상 = 안정, 0 이상 = 적정, -2 초과 = 상향, -2 이하 = 도전
• 영어가 1등급 부족하면 판정을 1단계 낮추고, 2등급 이상 부족하면 도전으로 처리합니다.
