# keyword quality audit guide

## 1. 상단 요약(summary)
- 검색 키워드의 탐구 가치가 한 문장 안에서 분명히 드러나는가?
- 단순 설명이 아니라 왜 이 주제가 좋은지 보이는가?

## 2. 탐구 확장 방향(directions)
- bullet 3개가 서로 다른 역할을 하는가?
- 같은 말을 반복하지 않는가?
- 실제 탐구 설계에 바로 쓸 수 있는가?

## 3. 생활기록부 문장(record sentence)
- 세특 느낌이 나는가?
- 키워드 설명에 그치지 않고 활동-분석-확장 흐름이 있는가?
- 너무 길거나 반복적이지 않은가?

## 4. 탐구 패턴 가이드(pattern guide)
- 패턴 설명이 이해되기 쉬운가?
- 실제로 탐구 설계에 도움이 되는가?

## 5. 우선 수정 대상 분류 기준
- revise: 전체 구조는 괜찮지만 문장 톤 수정 필요
- expand: 주제 자체는 좋지만 탐구 방향이나 예시가 더 필요
