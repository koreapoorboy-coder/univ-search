# 학과별 키워드 엑셀 → 공용 엔진 시드 데이터 정리

## 요약
- 원본 시트 수: 49개
- 정제 후 고유 키워드 수: 2809개
- 1차 우선 투입 키워드 수: 971개

## 1차 우선 투입 계열
- AI·소프트웨어
- 반도체·전자·전기
- 신소재·화학공학
- 환경·에너지
- 생명·의료

## 왜 이 축이 효율적인가
1. 학생 검색량이 높을 가능성이 큰 키워드가 많음
2. 탐구 설계로 바로 이어지기 쉬움
3. 과목/계열 매핑이 명확해서 추천 엔진 보정에 유리함
4. 상담 전환 데이터로 연결하기 좋음

## 바로 엔진에 넣어야 하는 데이터 우선순위
1. `keyword_library_phase1.json`
   - 검색 결과를 실제로 만들 핵심 시드
2. `keyword_alias_seed.json`
   - 검색 실패를 줄일 유사어/영문 입력 보정
3. `sheet_cluster_summary.json`
   - 어떤 학과/계열에서 키워드가 나왔는지 추적할 요약 데이터
4. `keyword_library_full_seed.json`
   - 전체 확장용 원본 시드

## 추천 작업 순서
1. phase1 데이터부터 엔진에 연결
2. 클릭/선택 로그를 통해 반응 좋은 키워드 우선순위 재정렬
3. 반응이 높은 키워드부터 탐구 설계 2~3개씩 붙이기
4. 검색 실패 키워드를 alias 또는 신규 키워드로 추가

## 참고
- 현재 JSON은 `학과 중심 → 키워드 중심`으로 뒤집은 시드 데이터임
- 아직 탐구 카드 본문은 비워져 있고, `recommended_plan_types`만 우선 생성해둠
- 다음 단계는 이 시드에 탐구 설계 템플릿을 붙여 실제 `keyword_library.json`으로 변환하는 작업임
