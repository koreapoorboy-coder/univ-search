# concept_competency_map_core_v1 사용 가이드

## 목적
과목별 역량축을 실제 추천 로직에 반영하기 위해,
대표 개념이 어떤 역량을 가장 강하게 드러내는지 실무형으로 연결한 파일이다.

## 원칙
1. 학과를 먼저 개념에 직접 붙이지 않는다.
2. 개념이 대표하는 교과 역량을 먼저 계산한다.
3. 학과는 선호 역량축에 가중치를 얹는 방식으로만 작동한다.

## 해석 방식
- primary_axes: 이 개념의 중심 역량
- secondary_axes: 함께 따라오는 보조 역량
- axis_weights: 추천 로직에서 사용할 상대 강도 (5가 가장 강함)

## 1차 적용 과목
- 통합과학1
- 통합과학2
- 공통수학1
- 공통국어1

2. helper에서 학과 직접 보정보다 역량 적합도를 먼저 계산
3. 4번 후속 연계축도 역량축을 통해 정렬되도록 수정
