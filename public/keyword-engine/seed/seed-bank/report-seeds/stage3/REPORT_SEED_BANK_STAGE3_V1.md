## 3. Seed별 활용 요약

### RPT-011 — 일상 자원 전환·미시경제 비용-편익 분석형
- 학생용 라벨: 일상 사례를 경제 개념으로 해석하고 환경적 편익까지 분석하는 유형
- 융합 유형: 생활경제 + 환경문제 + 행동경제학 + 정책 제안
- 문제 프레임: 버려지는 자원의 대체 활용이 가계 비용과 사회적 환경 비용을 동시에 줄이는가
- 분석 방법: 비용-편익 분석, 외부효과 해석, 대체재 교차탄력성, 행동경제학적 실험 설계
- 적합 과목: 경제, 통합사회, 사회문제 탐구, 환경
- 적합 전공: 경제학과, 행정학과, 환경정책학과, 사회학과, 지속가능경영 관련 계열
- 트리거: 매몰비용, 대체재, 외부효과, 비용-편익, 넛지, 순환경제, ESG, 환경비용
- 금지/주의: 생활 꿀팁 소개형으로 축소 금지 / 콜라 세정 효과를 과학적으로 과장 금지 / 수치와 참고문헌 임의 생성 금지

### RPT-012 — 웨어러블 생체신호·AI 패턴 인식형
- 학생용 라벨: 센서로 수집한 생체 신호를 AI가 해석해 실제 보조공학 문제를 해결하는 유형
- 융합 유형: 물리 센서 + 수학적 신호처리 + 딥러닝 + 의료공학
- 문제 프레임: 발성 장애 또는 소음 환경에서 말소리 없이 의도를 전달할 수 있는가
- 분석 방법: 시계열 데이터 처리, 주파수 분석, 패턴 분류, 단어 오류율 비교, 엣지 처리 지연 분석
- 적합 과목: 물리학, 미적분, 확률과 통계, 정보, 생명과학
- 적합 전공: 컴퓨터공학과, 인공지능학과, 의공학과, 전자공학과, 재활공학 관련 계열
- 트리거: EMG, FFT, DTW, 생체신호, 무음성 인식, 엣지컴퓨팅, 보조공학, 딥러닝
- 금지/주의: 단순 AI 음성인식 소개형 금지 / 생체데이터 보안 문제 누락 금지 / 실제 논문·수식 서지 임의 확정 금지

### RPT-013 — 교육정책·진로동기 심리 분석형
- 학생용 라벨: 진학 통계 뒤의 동기 구조를 심리학과 정책 이론으로 해석하는 유형
- 융합 유형: 교육정책 + 자기결정이론 + 공공재 + 진로정체성
- 문제 프레임: 우수 이공계 인재가 왜 의학계열로 이동하며 정책은 이를 어떻게 개선해야 하는가
- 분석 방법: 연도별 통계 비교, 동기 연속체 분류, 정책 효과의 심리적 경로 분석, 국제비교/설문 확장
- 적합 과목: 사회, 경제, 사회문화, 생활과 윤리, 심리학, 진로와 직업
- 적합 전공: 교육학과, 행정학과, 정책학과, 사회학과, 심리학과
- 트리거: 의대쏠림, 이공계기피, 자기결정이론, 내재적동기, 공공재, 교육정책, 진로정체성
- 금지/주의: 단순 의대 선호 비판형 금지 / 통계 출처·연도 임의 변경 금지 / 제재 정책을 무조건 효과적이라고 단정 금지

### RPT-014 — 공공정책·계층별 경험 차이 분석형
- 학생용 라벨: 같은 제도가 집단별로 다르게 작동하는 이유를 사회학 이론으로 분석하는 유형
- 융합 유형: 사회문화 + 노동경제 + 정치와 법 + 윤리
- 문제 프레임: 동일한 공공정책이 왜 사회적 위치에 따라 다른 결과를 낳는가
- 분석 방법: 집단 비교, 비용-편익 분석, 계층 위치 해석, 법 적용 범위 검토, 국제비교
- 적합 과목: 사회문화, 경제, 정치와 법, 생활과 윤리
- 적합 전공: 사회학과, 행정학과, 정책학과, 사회복지학과, 경제학과
- 트리거: 임시공휴일, 사회불평등, 이중노동시장, 자영업자, 계급갈등, 여가사회학, 분배정의
- 금지/주의: 찬반 여론 소개형 금지 / 모든 집단을 동일한 수혜자로 가정 금지 / 정책 효과를 평균값만으로 단정 금지

### RPT-015 — 구조불평등·통계모형 정책분석형
- 학생용 라벨: 사회 계층 문제를 통계 모형과 정책 설계로 연결하는 유형
- 융합 유형: 사회문화 + 확률과 통계 + 데이터사이언스 + 행정정책
- 문제 프레임: 법적 평등 이후에도 귀속 지위가 자원 배분에 어떤 영향을 미치는가
- 분석 방법: 변수 설정, 더미 변수화, 다중회귀분석, p-value 검정, 정책 타깃팅, 비교통계
- 적합 과목: 사회문화, 확률과 통계, 정치와 법, 데이터과학, 통합사회
- 적합 전공: 사회학과, 행정학과, 데이터사이언스학과, 국제학부, 정책학과
- 트리거: 카스트, 사회계층화, 할당제, 적극적우대조치, 다중회귀, 낙인효과, 데이터기반정책
- 금지/주의: 카스트를 단순 신분제 소개로 축소 금지 / 회귀분석 결과를 실제 검증 없이 확정값처럼 표현 금지 / 민감한 집단 데이터의 윤리 문제 누락 금지
