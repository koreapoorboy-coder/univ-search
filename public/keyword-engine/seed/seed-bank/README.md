# Report Seed Bank

Updated: 2026-05-14

이 폴더는 수행평가 탐구엔진 `keyword-engine`의 보고서 생성 품질을 높이기 위한 seed-bank입니다.

## 현재 누적 상태

- Stage 1: RPT-001~RPT-005 완료
- Stage 2: RPT-006~RPT-010 완료
- Stage 3: RPT-011~RPT-015 완료

총 seed 수: 15개

## 폴더 구조

```text
seed/seed-bank/
├─ report-seeds/
│  ├─ stage1/
│  ├─ stage2/
│  └─ stage3/
├─ raw-reports/
│  ├─ original-xlsx/
│  └─ converted-notes/
├─ index/
└─ README.md
```

## 중요

현재 seed-bank는 운영 엔진에 직접 연결하지 않는다.  
보고서 원문을 복사하지 않고, 보고서의 사고 구조와 전개 패턴만 추출해 축적한다.
